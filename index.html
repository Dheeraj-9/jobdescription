About Archer

Archer empowers organizations to manage multiple dimensions of risk on one platform with on‑premises and SaaS offerings. The Role

We are seeking a junior level Architect and Developer to help design, build, and evolve our core software systems. This is a dual role. You will support best practices, work with engineering teams, and ship high quality code while collaborating with product leadership and customers to build a next generation application.

Key Responsibilities

  • Support architecture and technical strategy for large scale distributed systems and core applications.
  • Incorporate AI capabilities where they improve efficiency, accuracy, or user experience.
  • Help build next generation data ingestion processes and tools.
  • Assist in operating test and production environments on AWS.
  • Work with DevOps pipelines and best practices.
  • Evaluate and introduce technologies, patterns, and tools that improve the system and developer productivity.
  • Contribute to decisions on cloud infrastructure, database design, API architecture, and data flow.
  • Support technical direction for key projects and ensure adherence to architecture principles.
  • Design and document scalable software solutions and system integrations.
  • Write maintainable code and perform reviews for critical components.
  • Ensure testing meets best practices and compliance requirements.
  • Clarify or expand requirements as needed.
  • Collaborate with cross functional teams on product roadmaps and technical milestones.

Required Qualifications

  • 3 years of professional software development experience.
  • 1 to 3 years in a software architect or engineer role.
  • Growth mindset and ability to adapt to changing priorities.
  • Hands on approach focused on team and company outcomes.
  • Strong proficiency in one or more modern languages such as Node.js, C#, JavaScript or TypeScript, or Python.
  • Experience with databases both SQL and NoSQL, message queues, and scaling data pipelines.
  • Expertise in system design, cloud native architecture on AWS, and distributed computing.
  • Deep understanding of API design, security, CI/CD, microservices, and modern DevOps practices.
  • Excellent communication and technical leadership skills.

Preferred Qualifications

  • Experience with SaaS and enterprise scale products.
  • Familiarity with container orchestration using Kubernetes and serverless.
  • Knowledge of architectural standards such as Domain Driven Design.
